El juego trata de una chica en un bosque encantador,que le persigue un zoombie feo,
si la chica toca al zoombie se muere o desaparece,para poder jugar este maravilloso y entretenido juego necesitas si o si descargar la app GAMER MAKER.
Este juego fue desarrollado con la herramienta GAME MAKER.
Puedes instalarlo simplemete con un espacio minímo en disco de 2g.
Ademas en la carpeta te proporsiona el ejecutable.
